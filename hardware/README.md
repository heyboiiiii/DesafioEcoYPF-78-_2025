YPF KICAD DESIGNS

Esquemas y PCB de KICAD para la computadora del auto del "Desafío ECO YPF" y sus submódulos.

Proyecto *AUTO*:
    
    Computadora basada en un ESP32, una pantalla led ILI9844 montada en el volante(SPI), un acelerometro(MPU6050), modulo GPS(NEO6M) y un modulo LORA(XL1278).  

Proyecto *SHIELD_LORA*:
    
    Adaptación del módulo SMT1278 LoRa a un conector de 5 pines (header de 2.54 mm).

Proyecto *ACS_V_placa*:
    
    Aquí es donde los terminales de las baterías se conectan al sensor de corriente ACS712 (30A) y al divisor de voltaje.

    Esta placa se conectará a la placa del Proyecto AUTO mediante 3 cables.

Project_libraries: librerias de simbolos y huellas de los componentes creados para adaptar cada modulo a la placa.
