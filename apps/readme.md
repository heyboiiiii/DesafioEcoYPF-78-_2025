# Pagina web 

# Esta se encarga de mostrar al usuario toda la informacion que puede sensar la computadora del auto y la envia a traves de un modulo Lora SMT1278.

# La pagina web es alojada internamente en un servidor montado con LittleFS sobre el esp32 conectado en boxes como receptor.

# Para acceder a esta se debe primero conectar al Access Point generado por el esp32 con las credenciales cargadas y entrar por un navegador a la direccion IP del esp32 por el puerto 80.
